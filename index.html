cocktail party

live

github

demo

For those who are big drinkers with poor memory. React based app that allows users to search through the CocktailDB API either by ingredient or alcohol name to generate drinks, their ingredients and recipe. Each drink also provides the top 3 closest LCBO locations based on user address input. Create an event and invite other users into the event via the unique React routed URL and let the drink adding begin. This project is collaborated with Thi Tran and Brent Carabott
